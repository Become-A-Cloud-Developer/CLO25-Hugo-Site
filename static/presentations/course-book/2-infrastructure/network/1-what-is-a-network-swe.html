Infrastrukturgrunder

Nätverkskonceptet

Ett nätverk ansluter datorenheter (noder) för att möjliggöra datautbyte och resursdelning

Data färdas i diskreta enheter som kallas paket som innehåller nyttolast och routinginformation

LAN (Local Area Network): enheter inom en byggnad/campus; WAN (Wide Area Network): täcker städer/kontinenter

Internet är ett globalt nätverk av sammankopplade nätverk som använder standardiserade protokoll

Viktiga nätverkskomponenter

Routers: Dirigerar data mellan nätverk med hjälp av IP-adresser; bestämmer optimala paketvägar

Switches: Hanterar data inom lokala nätverk baserat på MAC-adresser

Firewalls: Övervakar och kontrollerar trafik med hjälp av säkerhetsregler

Wireless Access Points: Bryggar trådlösa enheter till kabelansluten infrastruktur

Load Balancers & Reverse Proxies: Distribuerar trafik och förbättrar prestanda

Nätverkstopologier

Bus: Delad kommunikationslinje; kostnadseffektiv men begränsad feltolerans

Star: Enheter ansluter till central hubb; isolerar fel effektivt (dominerar moderna LAN)

Ring: Cirkulär anslutning; förutsägbar prestanda men sårbar för avbrott

Mesh: Flera redundanta vägar; maximerar tillförlitlighet men kräver mer infrastruktur

Nätverksabstraktionslager

Fysiskt lager: Påtaglig infrastruktur—kablar, nätverkskort, switches, routers

Virtuella nätverk: VLANs skapar logiska segment på delad fysisk infrastruktur

Molnnätverk: Azure VNets/AWS VPCs tillhandahåller isolerade miljöer via mjukvarudefinierad nätverksteknik

Serverlös nätverksteknik: Plattformen hanterar anslutning, lastbalansering och skalning automatiskt

Grundläggande nätverksfunktioner

Anslutning & routing: Etablerar vägar; routers vidarebefordrar paket hopp-för-hopp mot destinationer

Switching & segmentering: Switches vidarebefordrar inom LAN; VLANs isolerar trafik för säkerhet

Protokollstyrd kommunikation: TCP/IP-sviten ger grunden; HTTP, DNS, SSH bygger på toppen

Trafikhantering: Load balancers distribuerar förfrågningar; reverse proxies hanterar SSL-terminering och cachning

Nätverkssäkerhetsmekanismer

Firewalls: Filtrerar trafik baserat på IP-adresser, portar och protokoll

Network Security Groups: Molnbaserad filtrering på gränssnitts-/subnätsnivå

IDS/IPS: Övervakar för misstänkta mönster; varnar eller blockerar aktivt hot

Kryptering (TLS/VPNs): Skyddar data under överföring; skapar säkra tunnlar genom opålitliga nätverk

Viktig insikt

Nätverk tillhandahåller infrastrukturen som möjliggör kommunikation mellan enheter—att förstå routing, protokoll och säkerhetsmekanismer möjliggör välgrundade arkitekturbeslut
