Vad är nätverksförmedlare?

Enheter eller tjänster placerade mellan klienter och servrar för att hantera trafikflöde

Utför routing, lastdistribution, säkerhetsfiltrering, caching och infrastrukturabstraktion

Möjliggör funktioner som enskilda klienter/servrar inte kan koordinera själva

Grundläggande komponenter i produktionsnätverksarkitektur

Proxy-servrar

Förmedlar mellan klienter och externa servrar (utgående trafik)

Klientförfrågningar går först till proxy; proxy vidarebefordrar till målet och återger svaret

Funktioner: Tillgångspolicyhantering, innehållsfiltrering, caching, loggning, anonymitet

Klienter konfigurerar explicit för att använda proxyn; externa servrar ser proxy-IP, inte klient-IP

Reverse Proxies

Sitter framför backend-servrar och fångar upp inkommande klientförfrågningar

Fungerar transparent—klienter tror att de ansluter direkt till servern

Funktioner: Fördragsdistribution, SSL/TLS-terminering, caching, säkerhetsfiltrering

Ger stabilt gränssnitt medan backend-infrastrukturen ändras (lägga till/ta bort servrar)

Load Balancers

Distribuerar inkommande trafik över flera servrar för prestanda och tillgänglighet

Layer 4: Routar baserat på IP/port; snabb, minimal inspektion

Layer 7: Routar baserat på HTTP-innehåll (URL, headers, cookies); möjliggör sofistikerade regler

Om en server misslyckas routar load balancern till återstående friska servrar

Load Balancing-algoritmer

Round-robin: Cirkulerar genom servrar sekventiellt; enkelt, fungerar när servrar är likvärdiga

Least connections: Routar till server med minst aktiva anslutningar; hanterar varierande bearbetningstider

IP hash: Samma klient-IP når alltid samma server; stödjer sessionspersistens

Weighted: Tilldelar olika vikter baserat på serverkapacitet

Bastion Hosts

Ger kontrollerad åtkomst till servrar inom ett privat nätverk

Sitter i DMZ—mellan publikt internet och internt nätverk

Administratörer ansluter först till bastion, sedan till interna servrar därifrån

Härdad: Minimal mjukvara, strikta brandväggsregler, loggning, multifaktorautentisering

Azure Bastion: Hanterad tjänst som tillhandahåller webbläsarbaserad SSH/RDP utan att exponera VM:ar publikt

Viktig insikt

Nätverksförmedlare—proxies, reverse proxies, load balancers och bastion hosts—tillhandahåller väsentliga funktioner för säker, högpresterande och tillförlitlig infrastruktur
