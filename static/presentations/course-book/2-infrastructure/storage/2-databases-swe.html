DATABASER

Infrastruktur Grunder

Vad är en Databas?

Lagrar strukturerad data (ex. tabeller, dokument, tidsserier)

Gör det möjligt att söka i data (ex. SQL - Structured Query Language)

Upprätthåller dataintegritet (ex. Primary keys, foreign keys, not null, unique)

Hanterar relationer mellan data (1->n, n->n)

Relationsdatabaser

Använder tabeller med rader och kolumner

SQL (Structured Query Language) för datamanipulation

Starka ACID-garantier: Atomicity, Consistency, Isolation, Durability

Exempel: PostgreSQL, MySQL, SQL Server, Oracle

Bäst för strukturerad data med komplexa relationer och transaktionskrav

NoSQL-databaser

Designade för semistrukturerad data

Flexibla scheman för olika datatyper

Dokumentdatabaser: MongoDB, CouchDB - lagrar JSON-liknande dokument

Key-Value stores: Redis, DynamoDB - enkel nyckel-till-värde-mappning

Optimerade för horisontell skalning och hög genomströmning

In-Memory-databaser

Lagrar data i RAM för snabb åtkomst

Betydligt snabbare än diskbaserad lagring (mikrosekunder vs millisekunder)

Exempel: Redis, Memcached

Vanliga användningsområden: caching, sessionshantering, realtidsanalys

Tidsseriedatabaser

Speciellt designade för att hantera tidsstämplad data

Optimerade för loggar, mätvärden och sensordata

Exempel: InfluxDB, TimescaleDB, Prometheus

Använder effektiv komprimering för tidsseriemönster

Nyckelkoncept

Databas: Logisk container för data - innehåller data och metadata

Schema: Logisk struktur - definierar tabeller och relationer

Databasinstans: Den körande processen - själva programmet

Index: Snabbar upp datahämtning genom organiserad dataåtkomst

Databaser i Molnmiljöer

Managed Databases: Fullt hanterade av molnleverantörer (Amazon RDS, Azure SQL Database)

Leverantörer hanterar backuper, skalning och underhåll

Database as a Service (DBaaS): Förenklar driftsättning och hantering

Exempel: Firebase, Amazon DynamoDB, Azure Cosmos DB

🗄️

Sammanfattning

Databaser är ryggraden i moderna datadrivna applikationer. Att välja rätt typ och teknologi säkerställer skalbarhet, prestanda och säkerhet.
