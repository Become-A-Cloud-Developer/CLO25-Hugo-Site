# CLO25 Campus Mölndal - Class Roster

This is the master list of all students in the class. Use this as a reference when processing assignments.

## File Naming Convention

When renaming student files, use the prefix format: `lastname_firstname_`
- All lowercase
- Swedish characters simplified (ö→o, ä→a, å→a)
- Compound surnames joined without spaces
