# Docker

Learn container fundamentals through progressive exercises, from running your first container to orchestrating a multi-service development environment with Docker Compose.

## Learning Path

1. **Run Your First Container** — Pull, run, and manage containers with nginx
2. **Build and Push Your First Image** — Create a custom image and publish to Docker Hub
3. **Containerize CloudSoft Recruitment** — Single-stage and multi-stage Dockerfiles for a .NET app
4. **Docker Compose — Local Development Stack** — Orchestrate app + MongoDB with volumes and networking
5. **Multi-Platform Builds** — Build for both AMD64 and ARM64 with buildx
