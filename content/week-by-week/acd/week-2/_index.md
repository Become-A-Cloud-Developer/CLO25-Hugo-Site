# Week 2: Docker and Docker Compose

This week introduces containerization with Docker. You will learn the difference between containers and virtual machines, build Docker images for .NET applications, and use Docker Compose to run multi-service environments locally.

## Presentation

- **[Introduktion till Containers & Docker](/presentations/mini-lectures/introduction-to-containers.html)** — Theoretical introduction to containers, images, and why they matter for cloud development

## Theory

- Containers vs virtual machines — architecture and trade-offs
- Docker images, layers, and registries
- Multi-stage Dockerfile for .NET applications
- Docker Compose for multi-service local environments

## Practice

- Write a Dockerfile for an ASP.NET Core application
- Build and run containers locally
- Set up Docker Compose with MongoDB and Azurite

## Preparation

- Install Docker Desktop
- Read up on the difference between containers and virtual machines

## Reflection Questions

- What is the difference between a Docker image and a container?
- Why use multi-stage builds in a Dockerfile?
- How does Docker Compose connect multiple services in a local network?
