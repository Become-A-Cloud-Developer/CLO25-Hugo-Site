## Preparation

- Read up on GitHub Actions and YAML workflow syntax
- Make sure you can create a GitHub repo and run basic git commands:
  - `git clone <repo>`
  - `git add .`
  - `git commit -m "My commit"`
  - `git push`

## Reflection Questions

- What is the advantage of a CI/CD pipeline?
- What is the difference between Continuous Integration and Continuous Deployment?
- Which steps belong in CI and which belong in CD?
