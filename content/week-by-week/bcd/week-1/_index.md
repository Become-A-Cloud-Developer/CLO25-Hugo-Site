# Week 1 (v.5) — Virtual Servers

Course introduction. Create your Azure account, provision your first virtual machine, and connect to it via SSH.

## Theory

Read these chapters from the Course Book before or during the week:

- [Part I — Cloud Foundations](/course-book/1-cloud-foundations/) — what cloud computing is, service and deployment models, cloud-native development
- [Part II — Infrastructure: Compute](/course-book/2-infrastructure/compute/) — what a server is, server roles, physical vs virtual servers, Azure VM sizing

## Practice

- [Server Foundation — Portal Interface](/exercises/1-server-foundation/1-portal-interface/) — provision a VM through the Azure Portal, connect via SSH, and explore Linux from the command line

## Preparation

- Create a Microsoft Azure account
- Install a terminal/SSH client (built into macOS and Linux, available via OpenSSH or WSL on Windows)

## Reflection Questions

- What is a virtual server?
- What is Linux and how does it differ from Windows?
- How do you interact with a virtual server in the cloud via SSH?
