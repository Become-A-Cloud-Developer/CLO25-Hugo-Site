# Part III — Application Development

The application-development half of the BCD course rests on six load-bearing ideas: how HTTP carries requests and responses, what the .NET platform provides, how the MVC pattern organizes a web application, how the three-tier architecture separates responsibilities, how configuration travels with an application across environments, and how dependency injection wires the pieces together at startup.

The chapters in this Part run in reading order and build on each other. The companion exercise is the [Webapp Development chapter](/exercises/10-webapp-development/), which puts every concept into practice in a single growing ASP.NET Core MVC project.
