# Part X — Collaboration and Process

The Course Book closes where most jobs actually begin: in a team, on a board, with a Git remote that someone else also pushes to. Five chapters cover the practical mechanics — Git itself, the branching/PR/review workflow that keeps `main` healthy, the inner-loop vs outer-loop split that defines a developer's day, the agile vocabulary that organizes the work, and how all of it lands in Jira.

The companion exercise is the [Code Collaboration chapter](/exercises/15-code-collaboration/), which puts every concept into practice in a small team simulation.
